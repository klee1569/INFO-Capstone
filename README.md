# WellKare - a healthcare navigation Guide for New immigrants in King county

A web Application that is designed to help newly entered immigrants navigate the U.S healthcare system confidently

## Overview

Navigating the U.S healthcare system can be overwhelming for everyone, specially new immigrants. Many people struggle to understand how things work such as: health insurance, urgent care visits, appointments, where to find trusted healthcare providers

This project provides a simple and accessible web guide to helps users understand healthcare services, terminologies, and decision making through educational resources.

## Problem
New immigrants often face barriers when accessing healthcare, or struggle to understand the system including:

- Complex insurance process
- Difficulty medical terminology
- Language barrier
